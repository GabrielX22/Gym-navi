MANUAL TECNICO DE LA APLICACIÓN MOVIL

Este documento proporciona una guía detallada para facilitar la instalación del software desarrollado. Se subraya la importancia de prestar atención a los requisitos mínimos de hardware y software mencionados aquí para garantizar una instalación sin problemas del programa.

  • 1. REQUERMINIENTOS TECNICOS
    • 1.1. Requerimientos de hardware
      • Procesador: Qualcomm Snapdragon 420
      • Memoria RAM: 4gb o superior
    • 1.2. Requerimientos de software
      • Sistema Operativo: Android 7 o superior
  • 2. HERRAMIENTAS PARA EL DESARROLLO
    • 2.1. Android Studio
      • Se trata de un entorno de desarrollo integrado (IDE) empleado en la creación de aplicaciones destinadas a la plataforma Android. Además de contar con un editor de código y diversos componentes para el desarrollo, proporciona mejoras sustanciales en la productividad, convirtiéndolo en la herramienta preferida para el desarrollo de aplicaciones móviles.
  • 3. INSTALACION DE LA APLICACIÓN
    • Para la instalación del aplicativo móvil es necesario seguir los siguientes pasos, en primer lugar, descarga el apk que contiene el ejecutable de la aplicación esta estará disponible en la tienda de aplicaciones, y iniciar su instalación. Una vez realizado se instalará en el dispositivo.
  • 4.Arquitectura de la aplicación
    • Gymnavi está organizado en varios módulos principales, incluyendo Inicio de Sesión, Registro, Página Principal, Edición de Perfil, Definición de Objetivos y Rutina de Ejercicios. Cada módulo se encarga de una funcionalidad específica de la aplicación.
  • 5.Diagramas
    • Los diagramas de la arquitectura de la aplicación, el flujo de datos y los procesos se pueden encontrar en la documentación del proyecto en GitHub.
  • 6.Descripción de las funcionalidades
    • Cada módulo de la aplicación proporciona una funcionalidad específica. Por ejemplo, el módulo de Inicio de Sesión se encarga de autenticar a los usuarios, mientras que el módulo de Rutina de Ejercicios proporciona a los usuarios rutinas de ejercicios personalizadas.
  • 7.APIs y servicios externos
    • Gymnavi puede integrarse con varias APIs y servicios externos para proporcionar funcionalidades adicionales, como el seguimiento del progreso del usuario y la sincronización de datos entre dispositivos.
  • 8.Base de datos
    • Gymnavi utiliza una base de datos para almacenar información del usuario, como las rutinas de ejercicios, los objetivos de entrenamiento y el progreso del usuario.
  • 9.Código fuente
    • El código fuente de Gymnavi está escrito en Java y fue desarrollado en Android Studio. El código fuente está organizado de manera clara y coherente, y sigue las mejores prácticas de programación de Android.
  • 10.Pruebas
    • Se han realizado varias pruebas en la aplicación Gymnavi para asegurar su correcto funcionamiento. Esto incluye pruebas unitarias, pruebas de integración y pruebas de interfaz de usuario.
  • 11.Mantenimiento y soporte
    • El código fuente de Gymnavi está disponible en GitHub, lo que facilita su mantenimiento y actualización. Si encuentras algún problema con la aplicación, puedes abrir un issue en GitHub y nuestro equipo de soporte técnico se pondrá en contacto contigo.
  • 12.Licencia y términos de uso
    • Gymnavi es una aplicación de código abierto bajo la licencia MIT. Esto significa que puedes usar, copiar, modificar, fusionar, publicar, distribuir, sublicenciar y/o vender copias del software, siempre y cuando incluyas el aviso de copyright correspondiente y este aviso de permiso en todas las copias o partes sustanciales del software.
